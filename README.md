# 🎮 坦克大战 (Tank Battle)

经典坦克大战 HTML5 游戏，纯原生 JavaScript + Canvas 实现。

## 🕹️ 玩法

- **移动**: 方向键 ↑↓←→ 或 WASD
- **射击**: 空格键 或 J
- **开始/重来**: Enter

## 🎯 目标

保卫你的基地🏴，消灭所有敌人坦克！

- 共 **5 波** 敌军，每波敌人数量递增
- 玩家有 **3 条命**，死亡后短暂无敌重生
- 砖墙可被子弹摧毁，钢墙不可摧毁
- 基地被毁或生命耗尽则游戏结束

## 🚀 运行

直接用浏览器打开 `index.html` 即可。

## 🎨 特性

- Canvas 渲染，流畅 60fps
- 粒子爆炸特效
- AI 敌人（随机移动 + 追踪玩家）
- 波次递增难度
- 音效反馈
- 本地最高分记录
- 无敌重生保护
- 墙壁滑动碰撞优化
